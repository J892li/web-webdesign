POKE BOX Web Design

The Poke Box began serving their popular and delicious poke to Toronto’s financial district beginning the fall of 2016. Since then, they have established a great customer base, serving using ingredients that are superior in freshness, flavours, and nutrition.


create a web page for free
Poke box operate based on 2 pillars which are delivery and menu. Users spending time to view these sites because it is essential to gain information about food delivery and menu. It also motivates users to discover information about a specific product or service to help  they make decisions before dinning. 


Results：We coded 5 pages in the end, including 2 draft design, 2 times of user testing and finally over 80% of user satisfaction.

Solutions
  • User-centred design - 80% of users visit this website for checking menu and ordering food, we make sure our web is easy to read by using a clear structure.
  • Design for consistency and affordance - Avoid feature shock, using a consistent colour pattern, similar bowl shape to address the menu to create an authentic simulation of real poke box.
  • Fitts's law -Shortest screen distance/ most obvious shape for button design to reduce mind processing time
